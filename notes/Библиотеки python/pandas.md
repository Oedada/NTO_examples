 ### pandas - библиотека для работы с таблицами
## Концепция
Основными двумя сущностями в pandas являются Series и DataFrame(df), Series - одна колонка данных с именем, как колнка в таблице, а вот самой таблицей и является DataFrame.
## Обращение
```python
import pandas as pd

df = pd.DataFrame({
    "name": ["Иван", "Анна", "Олег"],
    "age": [20, 25, 22],
    "city": ["Москва", "Берлин", "Париж"]
}, index=["a", "b", "c"])
'''
   name  age    city
a  Иван   20  Москва
b  Анна   25  Берлин
c  Олег   22   Париж
'''
df["age"] --> Series
df["age"][0] --> int - 20
df.loc("a", "city") --> value(str) - Москва
df.loc("b") -- > Series - Анна 25 Берлин
df.loc("a":"c", ["age", "city"]) --> DataFrame (С первой по третью строку    !включительно! и две колонки age и city)
df.iloc(1, 2) --> value(str) - Берлин
'''
df.iloc() - работает ровно также, как и df.loc(), но обращается ко всему по индексам и пре срезе ведёт себя как списки python
'''
```
## Фильтрация и выборка
Суть в том, что при любых логических операциях pandas возращает маску на тот Series, над которым произоводилась операция, где лежат только True и False, на местах самих значений. Потом с помощью этой маски можно отобрать те значения, которые подходят под фильтр.
```python
import pandas as pd

df = pd.DataFrame({
    "name": ["Иван", "Анна", "Олег"],
    "age": [20, 25, 22],
    "city": ["Москва", "Берлин", "Париж"]
}, index=["a", "b", "c"])
mask = df["age"] > 21
'''
0 False
1 True
2 True
'''
df[mask]
'''
b  Анна 25  Берлин
c  Олег 22  Париж
id name age city
'''
mask = (df["age"] > 21) & (df["city"] == "Париж") # побитовое сравнение обяз.
'''
0 False
1 False
2 True
'''
df[mask]
'''
c  Олег 22  Париж
id name age city
'''
```
